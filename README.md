Photography Booking Website

A simple, modern static website for booking photography sessions, built with HTML and Tailwind CSS.

Features

*   Clean, Modern Design: Features a sleek black and white theme with a vibrant conic gradient accent.
*   esponsive Layout: Built with Tailwind CSS for adaptability across devices.
*   Clear User Journey: Outlines the 3-step booking process (Match, Show Up, Receive Photos) with numbered steps and imagery.
*   Strong Call to Action: Prominent "book a photographer now" button to drive conversions.

Getting Started

This is a static website. To view it locally:

1.  Clone or download this repository.
2.  Open the `index.html` file in your web browser.

Technologies Used

*   HTML5
*   Tailwind CSS (Output file included at `./src/output.css`)
*   Boxicons (via CDN for icons)

## Project Structure

```
project-root/
├── index.html
├── img/                 # Directory for image assets
├── src/
│   └── output.css       # Compiled Tailwind CSS
└── README.md
```

Usage

This website is designed as a landing page to attract and convert potential photography clients. Replace the placeholder images (`/img/Family-Gallery-1080x1080-2.webp`, etc.) and text with your own content and branding.
